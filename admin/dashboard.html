Logs hệ thống

Log container Docker của Platform (hoặc file cấu hình trên server).

Chọn tab và bấm Làm mới để tải log.

Mặc định: docker logs với tên container = hostname trong container (hoặc PLATFORM_DOCKER_LOGS_CONTAINER). Nếu không có Docker trong container: đặt PLATFORM_SYSTEM_LOG_PATH trỏ tới file log để đọc phần cuối.
